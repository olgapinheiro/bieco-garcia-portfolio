# Bieco Garcia - Photographer

This project features the portfolio of Bieco Garcia, a talented brazilian photographer mainly focused in the hospitality industry.

The website was built prioritizing a clean minimalist design, great responsiveness and performance.

It's currently not open to contributions, though you can fork this project for learning purposes and reference the original work, of course ;)


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
